# 🚀 mirrorMind
**A Utility-Based Emotional Reflection Agent**

## 📖 Project Overview
mirrorMind is an AI companion designed to help children in Karachi regulate their emotions through real-time feedback and guided reflection.

## 🛠️ Tech Stack
- **AI/CV:** DeepFace & OpenCV
- **Logic:** Python-based Utility Engine
- **Storage:** CSV Data Logging

## 🧠 System Architecture (Week 5 Progress)
We have implemented a **Utility-Based Agent** that goes beyond simple reflection. 
- **The Brain:** Calculates a Wellness Index based on a 15-frame buffer.
- **Intervention:** Automatically triggers Pluto's reflection prompts when low utility is detected.
